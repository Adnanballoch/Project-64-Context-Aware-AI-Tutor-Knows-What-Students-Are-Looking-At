# Project 64: Context-Aware AI Tutor — Knows What Students Are Looking At

**Category:** EdTech
**Part of:** AutoMatrixLab — 100+ AI and automation apps

## Overview

An AI tutor that tracks exactly what a student is looking at — the video frame being watched, the text selected, the code line they are stuck on — and responds with precise references: the exact lecture page, specific practical instruction, relevant code lines, and video timestamp where the professor explained the concept.

## Tech Stack

RAG, Vision AI, Video Processing, LLM, Python
